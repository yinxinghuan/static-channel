Static Channel — UI States 2026-06-01 v0.2 · 一页看全游戏每个阶段长什么样 · 真实 CSS, 真实文案

每张是一个 mini app 全栈 (header + ticker + TV + dial + stay-on-air + keep + 进度), 320px 宽, 完全按 src/StaticChannel/*.tsx + *.less 抓的样式. 频道图用 SVG/CSS 模拟 VHS 配色 (实际是 wdabuliu API 出的图).

状态机大致: 首次打开 → 拖动 → 停下设值 → 频率从未生成过 ⇒ 自动调用 gen-image, 中间 caption 走 meta / image 相. 频率已有 broadcast (自己或别人) ⇒ 直接接进, 不重生. 之后就是 keep ♡ / Stay On Air / 翻 Wall 看别人 / 红点提醒.

KEPT tab 只显示你 bookmark 的频率. 有新段的两张获得 橘红描边 + glow + 右上角红点脉动 — 别人在你的电台上加了新段. 点进 → 调到那个频率, lastSeenTs 自动推进 → 红点清掉。
不在这页里的, 说明清楚:
· 声音 — 拖动时白噪声音量跟速度走, 锁台后 hiss = 0. 没做 audio test 页因为这游戏不需要 TTS。
· 真实生图 — 每个频道的图是 wdabuliu API 出的, 风格在 prompts.ts 里钉死: "shot on degraded VHS tape, scan lines, CRT phosphor glow, slight chromatic aberration, 4:3 aspect, washed faded colors, no text". 这里用 CSS gradient 替代示意。
· 动画 — TV 雪花 100% 噪声 (这里 CSS 模拟得粗); ticker 圆点 2.4s 呼吸; 红药丸 1.6s 缩放; 新段红点 1.5s 脉动. 全跟生产一致。
· 缩放比例 — 实机 460px 宽, 这里 320px (~70%), 所有字号比例放大了一点点防止过小. dial 刻度数量也减了。
